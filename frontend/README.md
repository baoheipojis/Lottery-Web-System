## 开发设置

### 配置本地环境
1. 复制示例配置文件作为本地配置：
```bash
cp vue.config.js.example vue.config.js
```

2. 根据你的本地开发环境修改 `vue.config.js` 文件中的配置，例如API服务器地址等。
